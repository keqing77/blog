## 创建项目

### Vite

> Vite 创建 Vue 项目

```bash
npm init vue@latest
```

### Vue-cli

> Vue 脚手架 创建 Vue 项目

```bash
npm create vue@3
```

```bash
npx nuxi init `nuxt-app`
```
